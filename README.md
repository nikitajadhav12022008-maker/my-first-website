## 🌸 My First Website

This is my very first website project, created to learn **HTML**, **CSS**, and **JavaScript**.  
It’s a beginner-level project to practice building a web page and connecting scripts.

---

## 💻 Project Features

- Beautiful, simple layout with **HTML & CSS**  
- Interactive **button** using JavaScript  
- Personal “Coding Journey” section  
- Responsive design for beginners  
- Fun and colorful design 🌈  

---

## 📂 Files in the Project

| my_first_website| contains 3 main files index.html,style.css and script.js|
|-----------------|-----------------------------------------|
| index.html      | Main HTML file containing the website structure |
| style.css       | CSS file for styling the website        |
| script.js       | JavaScript file for button interaction  |

---

## 🚀 How to Run Locally

1. Make sure you have a web browser (Chrome, Edge, Firefox) installed.  
2. Open the project folder.  
3. Double-click on **index.html**  
4. The website will open in your browser.  
5. Click the button to see JavaScript in action!  

---

## 🌍 Optional: Make It Live Online

You can deploy this website on **GitHub Pages**:

1. Push your project to a GitHub repository.  
2. Go to **Settings → Pages → Source → main branch / (root)**  
3. Save and wait 30–60 seconds.  
4. Your live website link will appear — share it with friends!
